# Лабораторна робота №4: Візуалізація даних та обробка сигналів
**Виконала**: Любочка Вікторія  
**Група**: ФБ-44 
## Опис проєкту
Ця робота присвячена розробці інтерактивного інструменту для аналізу гармонійних сигналів.
Програма дозволяє моделювати фізичні процеси через побудову графіків, накладання керованого шуму та застосування цифрових фільтрів для відновлення чистого сигналу в реальному часі. 

## Виконані завдання
1. **Математичне моделювання**: Реалізовано функцію harmonic_with_noise для генерації сигналу `y(t) = A*sin(ωt +φ))`. 
2. **Інтерактивний GUI**:
 * `Слайдери`: Керування амплітудою, частотою, фазою та параметрами шуму (середнє значення та дисперсія).
 * `Чекбокси`: Перемикання відображення зашумленого сигналу та результату фільтрації. 
 * `Кнопка Reset`: Миттєве повернення всіх налаштувань до вихідного стану. 
  
3. **Оптимізація обчислень**: Впроваджено логіку збереження стану шуму, завдяки якій він не перегенеровується при зміні параметрів самої гармоніки.  
4. **Цифрова обробка сигналів**: Інтегровано IIR-фільтр низьких частот (фільтр Баттерворта) для ефективного придушення завад.  
5. **Динамічна візуалізація**: Налаштовано автоматичне оновлення осей та графіків при взаємодії з віджетами.  

## Технічні вимоги та запуск
1. Версія інтерпретатора: Python 3.10 або новіше.  
2. Встановлення залежностей: `pip install -r requirements.txt`.  
3. Запуск: Відкрийте файл `lab4.ipynb` у середовищі Jupyter Notebook або VS Code.  
4. Примітка: Для коректної роботи інтерактивних вікон у Windows використовуйте бекенд `%matplotlib qt`. 

## Інструкція користувача
* Використовуйте ліві слайдери для налаштування форми сигналу.  
* Використовуйте праві слайдери для зміни інтенсивності шуму.  
* Слайдер "Зріз фільтра" допомагає знайти оптимальну частоту для очищення сигналу.  
* Через панель чекбоксів можна порівнювати чисту гармоніку, зашумлену версію та результат роботи фільтра.

# Лабораторна робота №2: Наука про дані: підготовчий етап

**Виконала:** Любочка Вікторія ФБ-44

## Опис завдання
1. **Частина 1:** *Аналіз індексів VHI (Vegetation Health Index)*
У цій частині реалізовано обробку часових рядів супутникових даних для областей України:

* Завантаження та очищення: Автоматичне зчитування CSV-файлів, видалення системних тегів NOAA та обробка пропущених значень.
* Переіндексація: Створення словника (id_map) для приведення оригінальних ID областей у відповідність до українського алфавітного порядку.

* Функціонал вибірок:
  * Пошук значень VHI для конкретної області за вказаний рік.
  * Пошук екстремальних значень (min/max) для обраного регіону за діапазон років.
  * Фільтрація областей за рівнем посухи (наприклад, VHI < 15).

2. **Частина 2:** *Аналіз енергоспоживання домогосподарств*
Робота з великим датасетом (Individual Household Electric Power Consumption) з акцентом на продуктивність:

* Data Cleaning: Обробка великого обсягу даних, заміна символів ? на NaN та перетворення типів даних у float64.
* Складні фільтрації:
  * Відбір записів із активною потужністю > 5 кВт.
  * Визначення періодів, коли споживання пральної машини та холодильника перевищувало споживання кліматичних систем при силі струму 19-20 А.
  * Реалізація багаторівневого фільтра (час > 18:00, потужність > 6 кВт) із кроковим вибором результатів (кожен 3-й та 4-й запис).
* Математична обробка:
  * Обчислення коефіцієнтів кореляції Пірсона та Спірмена між напругою та потужністю.
  * Нормалізація даних(Min-Max Scaling) та Стандартизація.
  * Кодування категоріальних ознак (One-Hot Encoding) для типу дня будень/вихідний.
  * Профілювання: Використання модуля timeit для аналізу швидкості виконання кожної процедури фільтрації.

# Лабораторна робота №2: Наука про дані: підготовчий етап

**Виконала:** Любочка Вікторія, група ФБ-44

## Опис проекту
Ця робота присвячена первинній обробці, очищенню та статистичному аналізу даних з використанням бібліотеки **Pandas**. Основна увага приділена ефективності коду та нормалізації ознак для подальшого машинного навчання.

## Структура папки `lab2/`
* `Lab_2_Part1.ipynb` — обробка та алфавітна переіндексація даних ДМС (VHI) для регіонів України.
* `Lab2_Part2.ipynb` — аналіз споживання електроенергії, фільтрація та профілювання часу.
* `requirements.txt` — список необхідних бібліотек для запуску коду.
* `.gitignore` — локальні правила ігнорування важких датасетів (`.txt`, `.csv`).

## Виконані завдання (Part 2)
1. **Очищення даних:** Завантаження датасету `household_power_consumption`, обробка пропущених значень та конвертація типів.
2. **Фільтрація:** - Вибірка записів з активною потужністю > 5 кВт.
   - Пошук інтервалів сили струму 19-20 А з порівнянням споживання побутових приладів.
3. **Статистичний аналіз:** Обчислення середніх величин для випадкової вибірки (500,000 записів) та розрахунок коефіцієнтів кореляції Пірсона і Спірмена.
4. **Нормалізація та кодування:** - Застосування `MinMaxScaler` та `StandardScaler`.
   - Виконання One-Hot Encoding для категоріального атрибута `Day_Type`.
5. **Профілювання часу:** Аналіз швидкості виконання процедур за допомогою модуля `timeit`.

## Як запустити
1. Переконайтеся, що встановлено Python 3.8+.
2. Встановіть залежності: `pip install -r requirements.txt`.
3. Запустіть Jupyter Notebook або VS Code для перегляду `.ipynb` файлів.

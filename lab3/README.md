## Опис проекту
Ця робота присвячена дослідженню взаємозв'язків у багатовимірних наборах даних на прикладі датасету **Auto MPG**. Основна увага приділена побудові змістовних візуалізацій, що відображають залежності між технічними характеристиками автомобілів та їхньою енергоефективністю, згідно з принципами сучасного візуального аналізу.

## Структура папки `lab3/`
* `lab_3.ipynb` — основний Jupyter Notebook з очищенням даних та розширеною візуалізацією.
* `auto-mpg.data` — датасет з характеристиками автомобілів 1970-1982 рр.
* `requirements.txt` — список необхідних бібліотек (`seaborn`, `matplotlib`, `pandas`, `numpy`, `scipy`).
* `.gitignore` — локальні правила ігнорування тимчасових файлів Jupyter та кешу Python.

## Виконані завдання
1. **Підготовка даних:**
  - Завантаження: Використання pandas.read_csv з регулярним виразом sep=r'\s+' для коректної обробки пробілів у сирих даних та ігнорування коментарів (comment='\t').
  - Очищення: Ідентифікація пропущених значень (символ ?) та їх заміна на медіану за допомогою fillna(). Це дозволило зберегти цілісність вибірки для коректного розрахунку кореляцій.
  - Типізація: Примусове приведення ознаки horsepower до числового типу (float) для можливості математичної обробки.2. 
2. **Базова візуалізація:**
  - Гістограма: Побудова розподілу за 5-ма інтервалами з фіксованими межами [8, 16, 24, 32, 40, 48]. Додавання текстових анотацій (plt.text) для відображення точної кількості об'єктів у кожному біні.
  - Line Plot: Візуалізація часових трендів паливної ефективності. Використання агрегації за середнім значенням для кожного року з відображенням довірчих інтервалів, що дозволило простежити прогрес економності авто з 1970 по 1982 рік.3. 
3. **Багатовимірний аналіз (Medium-style):**
  - 4D/5D Scatter Plot: Дослідження взаємозв'язку між вагою, потужністю та MPG. Додавання додаткових вимірів через семантичне кодування: hue (колір — регіон походження), size (розмір — вага) та style (форма — кількість циліндрів).
  - FacetGrid: Розбиття складного графіка на декілька панелей (relplot з параметром col='origin'). Це дозволило локалізувати інженерні тренди окремо для американського, європейського та японського ринків.4.
4. **Статистична візуалізація:**
  - Heatmap: Розрахунок матриці кореляцій Пірсона для всіх числових ознак. Теплокарта наочно підсвітила мультиколінеарність між вагою, об'ємом двигуна та потужністю.
  - Pairplot: Побудова попарних залежностей, де діагональ відображає оцінку щільності розподілу (KDE). Це допомогло виявити кластеризацію автомобілів за ознакою походження.
  - Violin Plots: Порівняння «форми» розподілу MPG у різних регіонах, що продемонструвало наявність викидів у американському сегменті та високу стабільність економності в японському.

4. Переконайтеся, що файл `auto-mpg.data` знаходиться в одній директорії з ноутбуком.
